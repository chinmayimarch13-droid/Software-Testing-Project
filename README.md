# Automation Testing of Login Page using Selenium

## Project Description
This project demonstrates automation testing of a login page using Selenium with Python. 
The script automatically opens a browser, enters login credentials, and verifies the login functionality.

## Tools Used
- Python
- Selenium WebDriver
- Chrome Browser
- PyCharm IDE

## Project Files
login_test.py – Selenium automation script for login testing  
test_cases.txt – Test cases for login functionality  
bug_report.txt – Example bug report  
test_result.png – Screenshot of test execution result  

## Test Scenario
Automate the login functionality of a web application.

Steps:
1. Open the login page
2. Enter username and password
3. Click login button
4. Verify login success

## Expected Result
User should be able to login successfully with valid credentials.
